Teste com flexbox

Para o flexbox existe as propriedades para os container e outras para os itens

Flex container

Para os containers, temos as seguintes propriedades:

Flex-direction: column, column-reverse, row, row-reverse;

Flex-wrap: nowrap, wrap, wrap-reverse;

Flex-flow: flex-direction / flex-wrap

Justify-content: space-around/between/evenly, center, end;

Alinhamento normal

Align-items: flex-start/end, center, stretch, baseline;

Align-content: ;

Alinhamento normal

align-content: normal;

Alinhamento posicional basico

Align-content não recebe valores à esquerda nem à direita;

align-content: start, center, end, flex-start/end;

Alinhamento da linha de base

align-content: baseline, first baseline, last baseline;

Gap: gap é o espaçamento de um item para o outro;
